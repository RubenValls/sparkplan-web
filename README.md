# SparkPlan

SparkPlan is a web application built with [Next.js](https://nextjs.org) that helps users capture their ideas and automatically generate structured plans using AI. Users will be able to log in with Google, submit their ideas, and receive a detailed plan stored directly in their Google Drive.  

This project is currently in early development and serves as the foundation for integrating AI workflows and automation with [n8n](https://n8n.io).

---

## Features (Planned)

- Google authentication for secure user sessions.
- Text area to submit ideas.
- AI-powered language detection and plan generation.
- Automatic saving of structured idea plans to the user's Google Drive.
- Responsive design ready for desktop and mobile devices.
- Future PWA capabilities for mobile installation.
